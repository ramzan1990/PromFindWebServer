PromID

PromID is a sequence-based human TSS predictor built using deep learning. Taking advantage of big promoter collection with experimentally validated TSSs generated by modern high-throughput techniques, PromID builds a deep learning model using sequence data as its input. Specifically, to reduce the false positive rate, we designed PromID to adaptively train the predictor by changing the distribution of samples in the training set based on falsepositive errors it made in the previous iteration. By increasing the number of difficult non-promoter sequences in the training set, we can encourage the predictor to learn deeper and less obvious features to recognize a promoter sequence.
